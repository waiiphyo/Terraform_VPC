## Prerequisites
```
You must installed terraform in your machine
```
## Clone the repository
```
git clone https://github.com/waiiphyo/Terraform_VPC.git
``` 

## Useful commands

**check terraform version**
```bash
terraform --version
```

**Terraform init and plan**
```bash
terraform init
terraform plan
```

**Terraform apply and destroy**
```bash
terraform apply
terraform destroy
```
